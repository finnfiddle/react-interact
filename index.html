Interact

Declarative REST data fetching for React

View on GitHub Download .zip Download .tar.gz

Declarative

The days of communicating with your data store using an imperative API are over. Simply declare your data requirements as CRUD endpoints and let Interact fetch and mutate your data accordingly.

Colocation

Queries live next to the views that rely on them, so you can easily reason about your app. No more complex data stores and extensive plumbing needed.
